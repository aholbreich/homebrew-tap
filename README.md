# aholbreich Homebrew tap

Homebrew formulae for tools maintained by [@aholbreich](https://github.com/aholbreich).

## Install tl

```sh
brew install aholbreich/tap/tl
```

Or tap once and install by formula name:

```sh
brew tap aholbreich/tap
brew install tl
```

Install the current `main` branch from source:

```sh
brew install --HEAD aholbreich/tap/tl
```

## Available formulae

- `tl` — Git-native task ledger for human and AI agent coordination.
